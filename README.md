# Business Expense Tracker — Excel Dashboard

A professional financial tracker built for Indian SMEs and freelancers.
Tracks income, expenses, GST, vendors, and budget — all in one Excel file.

## Features

- Automatic Transaction ID generation
- GST % tracking per transaction
- Budget vs Actual with status indicators (Safe / Warning / Over)
- Vendor-wise expense and income analysis
- Monthly Cash Flow statement
- Profit & Loss summary
- Category and Payment Mode slicers for filtering
- Data validation dropdowns throughout
- Supports: UPI, Bank, Credit Card, Cash, Cheque, Net Banking

---

## Sheets Included

| Sheet | Purpose |
|---|---|
| Dashboard | Visual KPIs, charts, and slicers |
| Transactions | Main data entry sheet |
| Categories | Expense category and sub-category list |
| Vendors | Auto vendor analysis |
| Budget | Monthly budget planner |
| Cash Flow | Monthly income vs expense flow |
| P&L | Profit and Loss statement |
